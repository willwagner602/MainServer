About Me

I'm a programmer in San Jose, with lots of experience writing Python. I currently work at the KIPP Foundation in San Francisco, developing data automation for hundreds of schools across the country. Check out some of my projects below, and feel free to contact me via gmail or github.

PyJobs

PyJobs uses a group of RESTful APIs to aggregate jobs from a group of popular job search sites.

Thu2D

Thu2D is a javascript implementation of Terry Pratchett's Thud, a chesslike fantasy game. It's powered by a Python RESTful API.

Thud

Thud is the python backend supporting Thu2D. Documentation of the RESTful API can be found on its GitHub Page.
